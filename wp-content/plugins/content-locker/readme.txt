== Installation ==

1. Upload `content-locker.zip` to the `/wp-content/plugins/` directory and extract it
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create your own Lockers from the WordPress Dashboard using 'Content Locker' Menu
